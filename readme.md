# Galav - Social Image Sharing

## Quickstart

```sh
# 1. configure config/database.php
# 2. configure config/custom.php
# 3. create db
php artisan migrate
# 4. populate db (optional)
php artisan db:seed
```

## Done
Core functionality:
- Browsing images
- Upload of images
- User profiles page
- Tags
- Gif support
- Resizing images
- Voting
- Reporting/Flaging
